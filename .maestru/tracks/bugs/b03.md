# B03: Corrigir class= para className= no ItemMobile

## Problema

Em `src/components/itemMobile/ItemMobile.js`, o elemento `<table>` usa o atributo HTML `class=` em vez de `className=`, que é o atributo correto em JSX. Isto gera um warning do React e o estilo pode não ser aplicado.

```jsx
// Código atual — INVÁLIDO em JSX
<table class={classes.table_mobile}>
```

## Solução

```jsx
<table className={classes.table_mobile}>
```

- `src/components/itemMobile/ItemMobile.js`
