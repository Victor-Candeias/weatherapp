# B02: Adicionar key nas listas map() em Body.js

## Problema

Em `src/components/body/Body.js`, os componentes `ItemBrowser` e `ItemMobile` renderizados via `.map()` não têm a prop `key`, gerando warnings do React no console.

```js
// Código atual — sem key
{dailyMeteorology.data.map((day) => (
  <ItemBrowser value={day} ... />
))}
```

## Solução

Adicionar `key` única a cada item (usar `forecastDate` como identificador):

```js
{dailyMeteorology.data.map((day) => (
  <ItemBrowser key={day.forecastDate} value={day} ... />
))}
```
